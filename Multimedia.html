Éste espacio está diseñado para iniciarse en el mundo del diseño web, proporcionando información de lo más básico y fundamental para la creación de sitios web.

¿Qué es HTML?

HTML es el lenguaje de marcado usado para la creación de sitios web, en este se define una estructura básica para la construcción y presentación de su contenido, el cual está estandarizado gracias a W3C o World Wide Web Consontium fundado en 11994 por Tim Berners-Lee quien es también el creador del mencionado lenguaje.
